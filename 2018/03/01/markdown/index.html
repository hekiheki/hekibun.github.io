Markdown快速入门

Mar 1, 2018 | 基础

记录常用的Markdown的常用语法

注：Markdown使用#、+、-、*等符号标记时，符号后面至少要打一个空格才有效，除了常用符合外，还有一些html标签可以使用

标题

Markdown标题支持两种形式

  1. 用#标记

    在标题开头加上1~6个#，一次代表一级、二级…六级标题

    # 一级标题
## 二级标题
### 三级标题
##### 四级标题
###### 五级标题
###### 六级标题

  1. 用=和-标记

    在标题底下加上任意个=代表一级标题，-代表二级标题

    一级标题
=======
二级标题
-------

效果如下：

一级标题

二级标题

三级标题

四级标题
五级标题
六级标题

列表

Markdown 支持有序列表和无序列表

  1. 无序列表

    用+，-，*来标记

    + Color
+ Shape
+ Size
    + Big
    + Small
        + Very small

- Color
- Shape
- Size
    * Big
    * Small
        + Very small

    效果如下：

    • Color
    • Shape
    • Size
      • Big
      • Small
        • Very small

  2. 有序列表

    用数字加英文句点.来标记

    1. Color
2. Shape
3. Size
    1. Big
    2. Small
        1. Very small
    1. Color
    2. Shape
    3. Size
      1. Big
      2. Small
        1. Very small

代码

用反引号 ` 或者Tab(4个空格)

  1. 行内代码用反引号 ` 来标记，可嵌入文字中

    `print("hello world")`

    效果如下：

    `print("hello world")`

  2. 代码块用三个连续的反引号或者Tab，或4个空格来表示,代码语法高亮在反引号后面加上空格和语言名称即可
